The AutofacAppBuilderExtensions type exposes the following members.

Methods
  NameDescription
DisposeScopeOnAppDisposing
Registers a callback to dispose an Autofac ILifetimeScope when the OWIN host.OnAppDisposing event is triggered. This is a convenience method that will dispose an Autofac container or child scope when an OWIN application is shutting down.
IsAutofacLifetimeScopeInjectorRegistered
Determines if the Autofac lifetime scope injector middleware is registered with the application pipeline.
UseAutofacLifetimeScopeInjector
Adds middleware to inject a request-scoped Autofac lifetime scope into the OWIN pipeline.
UseAutofacMiddleware
Adds middleware to both inject a request-scoped Autofac lifetime scope into the OWIN pipeline as well as add all middleware components registered with Autofac.
UseMiddlewareFromContainerT
Adds a middleware to the OWIN pipeline that will be constructed using Autofac
