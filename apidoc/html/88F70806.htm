Remarks

This method uses the default Enterprise Library configuration source from Create to retrieve the set of configured Enterprise Library components and dependencies. It takes this set of configuration and adds the appropriate registrations to the provided builder.

If you need to provide a specific configuration source from which the container should be populated, use the RegisterEnterpriseLibrary(ContainerBuilder, IConfigurationSource) overload of this method.

Examples

This example shows a simple setup of registering Enterprise Library dependencies with Autofac.

After the container is built, use the AutofacServiceLocator to set the default Enterprise Library container for any internal resolutions it needs to make.

C#
Copy
// Create the container and register EntLib configured dependencies.
var builder = new ContainerBuilder();
builder.RegisterEnterpriseLibrary();

// Register any other dependencies, then build the container and set
// the default EntLib service locator.
var container = builder.Build();
var autofacLocator = new AutofacServiceLocator(container);
EnterpriseLibraryContainer.Current = autofacLocator;
